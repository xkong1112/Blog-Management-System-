# Blog-Management-System
前端开发的小案例
## 内容
### 用户管理页面
登录成功之后进入管理页面  
在管理页面展示在当前系统中注册的用户  
实现对用户的增删改查操作  
点击新增用户, 可以跳转到用户信息的编辑页面  


### 文章管理页面
分页 -> 减轻服务器的压力  
评论 -> 登录的用户允许评论  
实现对文章的增删改查操作, 包括文件的上传操作  

### 用户名
个人资料 ->修改用户资料  
退出登录  

## 文件夹名称及其实现的功能
public -> 静态资源  
model -> 数据库操作  
route -> 路由  
views -> 模板
